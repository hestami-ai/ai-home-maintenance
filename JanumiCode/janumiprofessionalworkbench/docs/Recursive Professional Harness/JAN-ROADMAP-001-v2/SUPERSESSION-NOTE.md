# Supersession and Consolidation Note

`JAN-ROADMAP-001 v2` supersedes the original master roadmap draft and incorporates the practical controls needed from the later roadmap-governance package.

The prior `JAN-ROADMAP-002`, `JAN-ROADMAP-003`, and detailed Wave 0 templates may remain useful as optional investigation aids. They are no longer required as a separate governance bureaucracy where the Master Roadmap, Detailed Roadmap Standard, and living registers provide equivalent control.

The governing operating model is now:

```text
Normative corpus + Master Roadmap + live repository
    → coding-agent-generated Detailed Implementation Roadmap
    → implementation and evidence
    → gate and next roadmap
```
